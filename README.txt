Eclipse git plugin allows execute some commands without command line but in eclipse view.

 Project is not intended to replace egit, but it offers functionality which currently is missing
 or has performance problems in egit. For example at current time commit using egit takes more than
 minute on big project.

 Instructions
 1. Go to main menu window - Preferences - GGit and enter here path to git executable and
 working dir (git repository)
 2. Show view GGit Status View (CTRL+3 GGit)
 3. Status view displays current repository status and allows execute git commands on selected
 files in the view.
